# 🔍 Digital Forensic Tool

A Flask-based web application for digital forensics analysis. Upload files, extract metadata, detect suspicious activity in logs, and generate professional forensic reports in PDF format.

![Uploading Digitalforensictool_filetest.png…]()


## ✨ Features

- **File Analysis**: Upload any file and get detailed metadata
- **File Type Detection**: Uses `python-magic` to detect real file type
- **Hash Generation**: MD5, SHA1, SHA256 hashes for file integrity
- **Log Analysis**: Scan log files for suspicious keywords and IP addresses
- **PDF Report Generation**: Download professional forensic reports with `reportlab`
- **Web Interface**: Clean, responsive UI built with Flask

## 🛠️ Tech Stack

- **Backend**: Python 3, Flask
- **Libraries**: python-magic, reportlab, werkzeug
- **Deployment**: Replit
